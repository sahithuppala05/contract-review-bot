# 📄 AI Contract Review Bot

### AI Engineer Hiring Assignment – Task 01

Built using Streamlit (Frontend) + Flask (Backend) + Groq LLM

---

## 🚀 Overview

The **AI Contract Review Bot** automates legal document analysis using Large Language Models.  
It accepts a contract (PDF or raw text), extracts key clauses, identifies potential risks, and generates a structured, plain-English summary.

This project demonstrates:

- LLM API integration
- Prompt engineering
- Backend API design
- Structured JSON extraction
- Risk classification logic
- Clean frontend-backend separation

---

## 🏗 Architecture

Frontend: **Streamlit**  
Backend: **Flask API**  
LLM: **Groq (Llama 3 – 70B)**  
PDF Parsing: **pdfplumber**
